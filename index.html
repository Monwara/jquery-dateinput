jQuery dateInput

This is the demo page for jQuery dateInput plugin. For the complete documentation, visit the GitHub repository page.

Inline calendar example

Left uses sundayFirst:true (default), while the rigth one uses sundayFirst:false

Pop-up with autoClose and noPast enabled

Note that the calendar (while open) reacts to changes in the value of the text input as user types in it.

The code

$(document).ready(function() {
  $('#test1').dateInput({inline: true});
  $('#test2').dateInput({inline: true, sundayFirst: false});
  $('#test3').dateInput({noPast: true, autoClose: true});
});
